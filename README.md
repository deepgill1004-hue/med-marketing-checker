## 三大功能

### 1️⃣ 合規檢查
貼文字或上傳圖檔（自動 OCR 抓字），一鍵掃描台灣醫美法規紅線：
- 108 條紅線詞、35 條黃線詞、17 種結構違規、12 個 regex 模式
- 法源涵蓋醫療法 §84-87、化粧品衛安法 §10、藥事法 §65-70、醫療器材管理法 §41、再生醫療法、公平交易法 §21
- 雙模式切換（個人 IP / 診所廣告），嚴格度不同
- 違規詞高亮 + 法源說明 + 自動重寫建議

### 2️⃣ 主題自動生成爆款合規文案
輸入療程主題（如玻尿酸、肉毒、隆乳手術）+ 內容類型（7 選 1）→ 一鍵生成：
- **7 種內容類型**：FAQ 解答型、比較選擇型、痛點共鳴型、反向勸退型、衛教知識型、故事型、時間線體驗型
- **6 個平台獨立骨架**（不是裁切版）：FB / IG / Threads / 小紅書 / LINE OA / LinkedIn
- **LINE OA 完整起承轉合**：爆 Hook → 場景+真實數據 → 反轉真相 → 強互動鉤子
- **60+ 筆台灣醫美實證數據**自動嵌入，含 PubMed / PRS Global Open / Motiva 等可查證來源

### 3️⃣ 自動配圖
純前端 Canvas API，每篇文案自動產 2 張圖：
- **封面大字報** 1080×1080（爆款 Hook + 內容類型標籤）
- **重點資訊卡** 1080×1350（從文案抓 3-5 條重點，數字徽章呈現）
- 4 套配色：溫柔醫美 / 信任專業 / 高級感 / 潔淨科技
- 一鍵下載 PNG

## 整合麗得醫美整形外科診所

勾選「使用麗得醫美素材」即可生成手術文案，內含：
- **陳錫根院長履歷**（前三總整外副院長、學會前理事長、手術數千例）
- **5 大手術**：隆乳（魔滴／女王波）、眼袋（內開／外開）、眼皮（雙眼皮／提眼瞼肌）、拉皮（額／中下臉 SMAS）、莢膜攣縮處理
- **自動套診所廣告模式**（最嚴合規規則）
